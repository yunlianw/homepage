# 个人主页展示系统

一个优雅的个人主页展示系统，支持多主题切换、音乐播放器、天气显示、favicon生成等功能。

## ✨ 特性

- 🎨 **3个精美主题** - default_bento / tech_minimal / apple_dark
- 🎵 **音乐播放器** - 播放列表、进度记忆、自动播放
- 🌤 **天气显示** - wttr.in免费接口，无需API Key
- 🖼 **Favicon生成** - 文字生成图标、自定义颜色、上传支持
- 📝 **文章管理** - 后台CRUD、静态生成
- 🔧 **扩展系统** - Key-Value配置，模板自由调用
- 📦 **一键安装** - install.php向导，自动检测域名
- 🔐 **后台管理** - 密码修改、后台目录自定义

## 🚀 安装

1. 下载安装包，解压到网站目录
2. 访问网站自动跳转到 `/install/`
3. 按向导完成：环境检测 → 数据库配置 → 初始化 → 创建管理员
4. 安装完成后删除 `install/` 目录
